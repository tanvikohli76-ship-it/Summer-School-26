IoT Week 3

Programs included: 

1. Servo Motor Control using Potentiometer: 
This project controls the angle of a servo motor using a potentiometer. The analog input from the potentiometer is mapped to a range of 0–180 degrees.
Concepts Used: Analog Input (`analogRead`), Mapping values (`map()`), Servo motor control (`Servo.h`)

Tinkercad link: https://www.tinkercad.com/things/bSWVLQMqXpP-servo-motor-control/editel?returnTo=%2Fthings%2FbSWVLQMqXpP-servo-and-potentiometer

2. LCD Display using I2C: 
This project displays messages on a 16x2 LCD using I2C communication. It prints text on both rows and clears the screen in a loop.
Concepts Used: I2C Communication, LCD interfacing (`LiquidCrystal_I2C`), Display functions (`setCursor`, `print`, `clear`)

Tinkercad link: https://www.tinkercad.com/things/loMl3rAKwZK-lcd-i2c-display/editel?returnTo=%2Fthings%2FloMl3rAKwZK-i2c

3. UART Communication (Master-Slave): 
This project demonstrates communication between two Arduino boards using UART. One acts as a transmitter (master) and the other as a receiver (slave) to control an LED.
Concepts Used: Serial Communication (UART), TX/RX data transfer, Conditional statements

Tinkercad link: https://www.tinkercad.com/things/54ifYy6GLaX-master-and-slave/editel?returnTo=https%3A%2F%2Fwww.tinkercad.com%2Fdashboard%2Fdesigns%2Fall

4. Potentiometer based Level Indicator System:  
This is a multi-component system where a potentiometer controls LEDs, a buzzer and a servo motor. It simulates a level-based system like a water level indicator.
Concepts Used: Analog Input, Arrays and loops, Mapping values, Servo control, Sound generation (`tone()`)

Tinkercad link: https://www.tinkercad.com/things/052U4y90Vx7-potentiometer-based-level-indicator/editel?returnTo=%2Fthings%2F052U4y90Vx7-arduino-potentiometer-based-level-indicator

5. LED Brightness Control: 
This project controls the brightness of an LED using a potentiometer. The analog input is converted into a PWM signal to vary the LED intensity.
Concepts Used: Analog Input (`analogRead`), PWM Output (`analogWrite`), Mapping values (`map()`)

Tinkercad link: https://www.tinkercad.com/things/eatY1zS3U89-led-brightness-control/editel?returnTo=https%3A%2F%2Fwww.tinkercad.com%2Fdashboard%2Fdesigns%2Fall

6. Automatic Night Light (LDR):   
This project uses a Light Dependent Resistor (LDR) to detect ambient light and automatically turn an LED ON or OFF.
Concepts Used: Sensor input (LDR), Voltage divider, Conditional statements (`if-else`)

Tinkercad link: https://www.tinkercad.com/things/5pbENrv5FZw-automatic-night-light-ldr/editel?returnTo=%2Fthings%2F5pbENrv5FZw-ldr

7. Temperature Controlled System (TMP36): 
This project reads temperature using a TMP36 sensor and controls outputs based on the temperature value. It can be used to simulate temperature-based automation systems.
Concepts Used: Analog Input, Sensor calibration (voltage → temperature), Conditional logic, Real-world data processing

Tinkercad link: https://www.tinkercad.com/things/1rzpTI9HbDo-temperature-controlled-servo/editel?returnTo=https%3A%2F%2Fwww.tinkercad.com%2Fdashboard%2Fdesigns%2Fall
