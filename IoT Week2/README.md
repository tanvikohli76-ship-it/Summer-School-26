IoT Week 2
Programs included:

1. Fade in Fade Out of leds: 
This program controls multiple LEDs and gradually changes their brightness using PWM (Pulse Width Modulation). Each LED fades in and out sequentially, creating a smooth lighting effect.
Concepts Used:
analog write, Arrays, Nested loops, Brightness control

Tinkercad link: https://www.tinkercad.com/things/eBstg7SncDZ-fade-in-fade-out/editel?returnTo=https%3A%2F%2Fwww.tinkercad.com%2Fdashboard%2Fdesigns%2Fall

3. 4x4 LED matrix:  
This program controls a 4x4 LED matrix by activating one LED at a time using row and column scanning. It demonstrates how multiple LEDs can be controlled efficiently using fewer pins.
Concepts Used:
Row-column addressing, Matrix making, Nested loops, Digital output control

Tinkercad link: https://www.tinkercad.com/things/c1DXMZiyeqV-4x4-led-matrix/editel?returnTo=https%3A%2F%2Fwww.tinkercad.com%2Fdashboard%2Fdesigns%2Fall

5. 7-Segment Display Counter: 
This program displays numbers from 0 to 9 on a 7-segment display by controlling individual segments.
Concepts Used:
Arrays (2D arrays), Digital output, Display control logic, Function usage

Tinkercad link: https://www.tinkercad.com/things/1CNPYxZBjl3-seven-segment-display/editel?returnTo=https%3A%2F%2Fwww.tinkercad.com%2Fdashboard%2Fdesigns%2Fall

6. Reaction Time Game: 
This interactive program measures the user's reaction time. After random LED signals, the user must press a button when the green LED turns ON. The reaction time is calculated and displayed on the Serial Monitor.
Concepts Used:
Digital input (push button), Serial communication, `millis()` for time measurement, Randomization, Interactive system design

Tinkercad link: https://www.tinkercad.com/things/eFDC8XmhWA2-led-reaction-time-game-using-arduino/editel?returnTo=https%3A%2F%2Fwww.tinkercad.com%2Fdashboard%2Fdesigns%2Fall
